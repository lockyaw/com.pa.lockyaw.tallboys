# Tallboys Ridiculous Rebalance Mod 💥🛰️

A drastic, extreme rebalance mod for **Planetary Annihilation: TITANS**. 

Originally created for a specific friend group, this server mod fundamentally changes how the late-game plays out by essentially axing traditional Nukes and heavy Orbital-to-Ground bombardment. It turns game-ending weapons into cheaper, tactical tools, forcing more ground, naval, and air engagements.

## 🛠️ Balance Changes

### The Nuke Nerfs
* **Cheap & Tactical:** Nuke Launchers and Missiles are both **1/4th the price**.
* **Massive Damage Reduction:** Nuke damage is slashed to **1/5th**. They will basically only one-shot Tier 1 units (like bots) now. Range is unchanged.
* **Virtually Free Defense:** Anti-Nuke Launchers are **1/6th the price**, and the anti-nuke missiles are **1/8th the price**.
* *(Note: The Manhattan mobile nuke remains completely untouched!)*

### The Orbital Overhaul
* **General Orbital Nerf:** Orbit-to-Ground damage is reduced by a massive **60%** across the board. Orbit-to-Orbit combat remains exactly the same.
* **Umbrella (Anti-Orbital):** Damage against orbital targets is reduced by **40%**.
* **SXX (Orbital Laser):** Moves 25% slower and deals 10% less ground damage.
* **Helios (Titan):** The one exception! The Helios gets a **30% price reduction** and a **50% increase** to its ground damage.
* *(Note: The Catapult and Orbital Defense Satellites remain completely untouched).*

## 📥 Installation

**Method 1: In-Game Community Mods (Recommended)**
1. Launch *Planetary Annihilation: TITANS*.
2. Open the **Community Mods** manager from the main menu.
3. Search for `Tallboys Ridiculous Rebalance Mod`.
4. Click **Install** and ensure the mod is enabled.

**Method 2: Manual / Local Installation**
1. Download the `.zip` of this repository.
2. Extract the folder
3. Move the folder to your Planetary Annihilation server mods directory:
   * **Windows:** `%LOCALAPPDATA%\Uber Entertainment\Planetary Annihilation\server_mods`
   * **Mac:** `~/Library/Application Support/Uber Entertainment/Planetary Annihilation/server_mods`
   * **Linux:** `~/.local/Uber Entertainment/Planetary Annihilation/server_mods`
4. Enable the mod in the "Installed" -> "Local Mods" tab of the Community Mods manager in-game.

## ⚙️ Important Notes

* **Server-Side Mod:** Because this mod alters unit stats, it is a **Server Mod**. This means it only needs to be installed by the host. If you host a local multiplayer lobby, all of your friends will automatically download the balance changes when they join your game!
* **Compatibility:** This mod overwrites the base `.json` files for the affected units. It should be compatible with most other mods, but may conflict with other heavy balance overhauls that touch the exact same Nuke/Orbital files.
